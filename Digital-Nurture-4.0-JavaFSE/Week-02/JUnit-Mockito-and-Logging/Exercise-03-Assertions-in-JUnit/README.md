# JUnit Assertions

## Objective
This exercise exercises standard JUnit assertions. It tests variables and object state verification.

## Files
*   `AssertionsTest.java` – Tests verifying various JUnit assertion checks
*   `pom.xml` – Maven configuration file containing project dependencies

## Approach
I implemented various assertions such as assertEquals, assertTrue, and assertNotNull to evaluate conditions.

## How to Run
```bash
mvn clean test
```

## Result
Running tests checks the assertions and returns a successful build report.

## What I Learned
I learned how to choose the right JUnit assertion method to test expected values.
